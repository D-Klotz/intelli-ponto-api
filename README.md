# IntelliPonto
API do sistema IntelliPonto, este sistema processa e armazena os registros de entrada e saída de um funcionário em uma determinada empresa. 

# Ferramentas Utilizadas
- Java
- Spring Boot
- JPA e Spring Data
- FlyWay
- TravisCI
- Swagger
- jwt authentication
- EhCache
